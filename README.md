# my_business_app

Django-opdracht met een eenvoudige bedrijfseconomische formule in MVT-structuur.

> Django projectpackage: `my_business_app`

## Formule
- **Input 1:** omzet
- **Input 2:** inkoopkosten
- **Output 1:** brutomarge = omzet - inkoopkosten
- **Output 2:** margepercentage = (brutomarge / omzet) × 100
